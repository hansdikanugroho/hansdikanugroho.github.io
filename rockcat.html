Description

A Game Story Based with side scroller view , the game that has mechanics for sorting waste. I created this game with my friends, where I acted as the programmer using Unity to create systems like player movement, UI implementation, character animations, and the application of cutscenes. Mostly in this project I become a programmer system who responsibility make an all the system of game ,. We developed this game for about 4 months. The aim of this game is to highlight the importance of managing and organizing waste on Earth.
